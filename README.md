PHP Password Library Documentation
==================================

This repository contains the sources for the PHP Password Library documentation,
which can be built with [Sphinx](http://sphinx.pocoo.org/) or read online at
[phpasslib.readthedocs.org](http://phpasslib.readthedocs.org/en/latest/index.html).
